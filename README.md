# weather-bot
Telegram Weather Bot

## Installation

1. Clone repo
2. Create virtual environment
```bash
python3 -m venv venv
source venv/bin/activate
```
3. Install dependencies
```bash
pip install -r requirements.txt
```
4. Run
```bash
python main.py
```
